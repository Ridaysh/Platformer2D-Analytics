
# Platformer2D-Analytics

Этот репозиторий содержит Unity-проект с интеграцией GameAnalytics для отслеживания игровых событий и базовых метрик.

## Описание

Проект представляет собой простой 2D-платформер с реализованной системой аналитики.

В рамках проекта:

- Настроен GameAnalytics Dashboard
- Выполнена ручная инициализация SDK
- Реализована отправка различных типов событий
- Добавлены отладочные логи для контроля работы аналитики



## Архитектура аналитики

- `GAInitializer` — ручная инициализация SDK
- `AnalyticsService` — централизованный сервис отправки событий
- Интеграция событий в игровые триггеры и систему управления уровнем
